# Web App: Ferramenta de Análise de Dados de Veículos 👋

## 📋 Descrição do Projeto
Desenvolvimento de uma aplicação web interativa para a análise exploratória de dados (EDA) de anúncios de vendas de veículos. O objetivo foi criar uma ferramenta acessível que permitisse a visualização rápida de tendências de mercado, quilometragem e condições dos veículos através de filtros dinâmicos.

## 🎯 Objetivos Estratégicos
* Facilitar a visualização de grandes volumes de dados para usuários não técnicos.
* Proporcionar uma interface interativa para análise de correlação entre variáveis (ex: preço vs. ano do modelo).
* Demonstrar a capacidade de deploy de soluções de dados em ambiente de nuvem.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python.
* **Framework Web:** Streamlit.
* **Bibliotecas:** Pandas (manipulação) e Plotly Express (gráficos interativos).
* **Deploy/Hospedagem:** Render.

## 📉 Funcionalidades da Aplicação
1.  **Dashboard Interativo:** O usuário pode gerar histogramas e gráficos de dispersão ao clique de botões.
2.  **Visualização de Distribuição:** Histogramas dinâmicos para analisar a condição e o ano dos veículos anunciados.
3.  **Análise de Dispersão:** Gráficos interativos para identificar padrões entre a quilometragem (odômetro) e o preço de venda.
4.  **Filtros de Dados:** Interface limpa que permite alternar visualizações de acordo com a necessidade da análise.

## ✅ Resultados
* **Acessibilidade:** Transformação de um conjunto de dados bruto em uma ferramenta de BI funcional.
* **Portfólio Técnico:** Consolidação de competências em desenvolvimento de aplicações de dados e gerenciamento de ambientes virtuais para deploy.
